# Tether v0.1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the tether-cli daemon, raw-mode `tether attach` client, and PWA per `docs/specs/2026-04-25-tether-v0.1-design.md`, gated on Week 1 PoCs (§11) that prove the node-pty + JSONL architecture before committing the full 11-week build-out.

**Architecture:** 2-tier (daemon on user's server + PWA on phone, no relay). Daemon owns the PTY master fd via node-pty so the single-writer lock is enforceable at byte level. JSONL is the single source of truth — chat replay uses CC's embedded message UUIDs as stable event IDs. Transport: WebTransport primary + WebSocket co-equal fallback. Crypto: XChaCha20-Poly1305 AEAD + libsodium box. Auth: refresh-token family with 30s idempotent grace window.

**Tech Stack:** TypeScript 5.6, Node.js 20 LTS, node-pty, libsodium-wrappers (sumo), `ws`, vitest, React 19, Vite 6, pnpm workspace.

```typescript
export interface ByteSinkOptions {
  delayMs?: number;
}

export class ByteSink {
  private chunks: Buffer[] = [];
  private total = 0;
  private readonly delayMs: number;

  constructor(opts: ByteSinkOptions = {}) {
    this.delayMs = opts.delayMs ?? 0;
  }

  write(bytes: Buffer): void {
    this.chunks.push(Buffer.from(bytes));
    this.total += bytes.length;
  }

  async writeAsync(bytes: Buffer): Promise<void> {
    if (this.delayMs > 0) {
      await new Promise((r) => setTimeout(r, this.delayMs));
    }
    this.write(bytes);
  }

  collected(): Buffer {
    return Buffer.concat(this.chunks);
  }

  get byteLength(): number {
    return this.total;
  }
}
```

```typescript
export interface RingBufferOptions {
  capacityBytes: number;
}

/**
 * Fixed-capacity byte ring buffer. Writes never block; oldest bytes drop on overflow.
 * Used for "last screen" replay to newly attached terminal clients.
 */
export class RingBuffer {
  private readonly buf: Buffer;
  private readonly capacity: number;
  private writeHead = 0;
  private filled = false;
  private _totalWritten = 0;
  private _bytesDropped = 0;

  constructor(opts: RingBufferOptions) {
    if (opts.capacityBytes <= 0) throw new Error('capacityBytes must be > 0');
    this.capacity = opts.capacityBytes;
    this.buf = Buffer.alloc(this.capacity);
  }

  write(chunk: Buffer): void {
    this._totalWritten += chunk.length;
    if (chunk.length === 0) return;

    if (chunk.length >= this.capacity) {
      this._bytesDropped += this.size + chunk.length - this.capacity;
      chunk.copy(this.buf, 0, chunk.length - this.capacity);
      this.writeHead = 0;
      this.filled = true;
      return;
    }

    const before = this.size;
    const tailRoom = this.capacity - this.writeHead;
    if (chunk.length <= tailRoom) {
      chunk.copy(this.buf, this.writeHead);
      this.writeHead += chunk.length;
      if (this.writeHead === this.capacity) {
        this.writeHead = 0;
        this.filled = true;
      }
    } else {
      chunk.copy(this.buf, this.writeHead, 0, tailRoom);
      const remaining = chunk.length - tailRoom;
      chunk.copy(this.buf, 0, tailRoom);
      this.writeHead = remaining;
      this.filled = true;
    }
    const after = this.size;
    if (before + chunk.length > after) {
      this._bytesDropped += before + chunk.length - after;
    }
  }

  snapshot(): Buffer {
    if (!this.filled) return Buffer.from(this.buf.subarray(0, this.writeHead));
    const out = Buffer.alloc(this.capacity);
    const tail = this.buf.subarray(this.writeHead);
    const head = this.buf.subarray(0, this.writeHead);
    tail.copy(out, 0);
    head.copy(out, tail.length);
    return out;
  }

  get size(): number {
    return this.filled ? this.capacity : this.writeHead;
  }

  get totalWritten(): number {
    return this._totalWritten;
  }

  get bytesDropped(): number {
    return this._bytesDropped;
  }
}
```

```typescript
import { EventEmitter } from 'node:events';

export interface FanoutOptions {
  perClientHighWaterBytes: number;
}

type Sink = (chunk: Buffer) => void | Promise<void>;

interface Subscriber {
  id: string;
  sink: Sink;
  pending: Buffer[];
  pendingBytes: number;
  draining: boolean;
  lagging: boolean;
}

export interface DropEvent {
  clientId: string;
  reason: 'client_lag' | 'ring_full';
  droppedBytes: number;
}

export class Fanout extends EventEmitter {
  private readonly subs = new Map<string, Subscriber>();
  private readonly hwm: number;

  constructor(opts: FanoutOptions) {
    super();
    this.hwm = opts.perClientHighWaterBytes;
  }

  subscribe(clientId: string, sink: Sink): () => void {
    const sub: Subscriber = {
      id: clientId,
      sink,
      pending: [],
      pendingBytes: 0,
      draining: false,
      lagging: false,
    };
    this.subs.set(clientId, sub);
    return () => {
      this.subs.delete(clientId);
    };
  }

  broadcast(chunk: Buffer): void {
    for (const sub of this.subs.values()) {
      if (sub.pendingBytes + chunk.length > this.hwm) {
        if (!sub.lagging) {
          sub.lagging = true;
          this.emit('drop', {
            clientId: sub.id,
            reason: 'client_lag',
            droppedBytes: chunk.length,
          } satisfies DropEvent);
        }
        continue;
      }
      sub.pending.push(chunk);
      sub.pendingBytes += chunk.length;
      this.drain(sub);
    }
  }

  private drain(sub: Subscriber): void {
    if (sub.draining) return;
    sub.draining = true;
    void (async () => {
      while (sub.pending.length > 0) {
        const next = sub.pending.shift()!;
        sub.pendingBytes -= next.length;
        try {
          await sub.sink(next);
        } catch {
          // sink failures: drop subscriber silently in PoC; production wraps with error event
          this.subs.delete(sub.id);
          return;
        }
      }
      sub.draining = false;
      if (sub.lagging && sub.pendingBytes === 0) sub.lagging = false;
    })();
  }
}
```

```typescript
import * as nodePty from 'node-pty';

export interface PtyExitInfo {
  exitCode: number;
  signal: number | undefined;
}

export interface PtyHandle {
  readonly pid: number;
  onData(handler: (chunk: Buffer) => void): () => void;
  write(bytes: Buffer | string): void;
  resize(cols: number, rows: number): void;
  kill(signal?: string): void;
  readonly exit: Promise<PtyExitInfo>;
}

export interface SpawnPtyOptions {
  cwd?: string;
  env?: NodeJS.ProcessEnv;
  cols?: number;
  rows?: number;
}

export function spawnPty(
  command: string,
  args: string[],
  opts: SpawnPtyOptions = {},
): PtyHandle {
  const proc = nodePty.spawn(command, args, {
    name: 'xterm-256color',
    cols: opts.cols ?? 80,
    rows: opts.rows ?? 24,
    cwd: opts.cwd ?? process.cwd(),
    env: (opts.env ?? process.env) as { [key: string]: string },
  });

  const exitPromise = new Promise<PtyExitInfo>((resolve) => {
    proc.onExit(({ exitCode, signal }) => resolve({ exitCode, signal }));
  });

  return {
    pid: proc.pid,
    onData(handler) {
      const disp = proc.onData((s) => handler(Buffer.from(s, 'utf8')));
      return () => disp.dispose();
    },
    write(bytes) {
      proc.write(typeof bytes === 'string' ? bytes : bytes.toString('utf8'));
    },
    resize(cols, rows) {
      proc.resize(cols, rows);
    },
    kill(signal) {
      proc.kill(signal);
    },
    exit: exitPromise,
  };
}
```

```typescript
import { EventEmitter } from 'node:events';

export type LockHolder =
  | { kind: 'terminal'; deviceId: string }
  | { kind: 'mobile'; deviceId: string };

export interface LockManagerOptions {
  idleMs: number;
  clock?: () => number;
}

export type DenyReason = 'wrong_holder' | 'idle_required';

export interface LockResult {
  granted: boolean;
  holder: LockHolder | null;
  reason?: DenyReason;
}

export interface LockTransition {
  kind: 'grant' | 'release_idle' | 'release_explicit' | 'force';
  at: number;
  from: LockHolder | null;
  to: LockHolder | null;
}

export class LockManager extends EventEmitter {
  private current: LockHolder | null = null;
  private lastWriteAt = 0;
  private readonly idleMs: number;
  private readonly clock: () => number;

  constructor(opts: LockManagerOptions) {
    super();
    this.idleMs = opts.idleMs;
    this.clock = opts.clock ?? Date.now;
  }

  holder(): LockHolder | null {
    this.maybeAutoRelease();
    return this.current;
  }

  tryAcquireForWrite(by: LockHolder): LockResult {
    this.maybeAutoRelease();
    if (this.current === null || this.holderEquals(this.current, by)) {
      const previous = this.current;
      this.current = by;
      this.lastWriteAt = this.clock();
      if (previous === null) this.emitTransition({ kind: 'grant', from: null, to: by });
      return { granted: true, holder: by };
    }
    return { granted: false, holder: this.current, reason: 'wrong_holder' };
  }

  requestExplicit(by: LockHolder): LockResult {
    this.maybeAutoRelease();
    if (this.current === null || this.holderEquals(this.current, by)) {
      const previous = this.current;
      this.current = by;
      this.lastWriteAt = this.clock();
      if (previous === null) this.emitTransition({ kind: 'grant', from: null, to: by });
      return { granted: true, holder: by };
    }
    return { granted: false, holder: this.current, reason: 'wrong_holder' };
  }

  forceTakeover(by: LockHolder): LockResult {
    const previous = this.current;
    this.current = by;
    this.lastWriteAt = this.clock();
    this.emitTransition({ kind: 'force', from: previous, to: by });
    return { granted: true, holder: by };
  }

  release(by: LockHolder): void {
    if (this.current && this.holderEquals(this.current, by)) {
      this.emitTransition({ kind: 'release_explicit', from: this.current, to: null });
      this.current = null;
    }
  }

  private maybeAutoRelease(): void {
    if (this.current !== null && this.clock() - this.lastWriteAt > this.idleMs) {
      this.emitTransition({ kind: 'release_idle', from: this.current, to: null });
      this.current = null;
    }
  }

  private emitTransition(t: Omit<LockTransition, 'at'>): void {
    const trans: LockTransition = { ...t, at: this.clock() };
    this.emit('transition', trans);
  }

  private holderEquals(a: LockHolder, b: LockHolder): boolean {
    return a.kind === b.kind && a.deviceId === b.deviceId;
  }
}
```

```typescript
import type { LockHolder, LockManager } from '../session/lock.js';

export interface InputGateOptions {
  lockManager: LockManager;
  write: (bytes: Buffer) => void;
}

export interface GateInput {
  sender: LockHolder;
  bytes: Buffer;
}

export interface GateDecision {
  outcome: 'written' | 'rejected';
  reason?: 'wrong_holder';
  banner?: string;
}

export class InputGate {
  constructor(private readonly opts: InputGateOptions) {}

  submit(input: GateInput): GateDecision {
    const result =
      input.sender.kind === 'terminal'
        ? this.opts.lockManager.tryAcquireForWrite(input.sender)
        : this.opts.lockManager.requestExplicit(input.sender);

    if (result.granted) {
      this.opts.write(input.bytes);
      return { outcome: 'written' };
    }

    const heldBy = result.holder!;
    return {
      outcome: 'rejected',
      reason: 'wrong_holder',
      banner: bannerFor(input.sender, heldBy),
    };
  }
}

function bannerFor(sender: LockHolder, holder: LockHolder): string {
  if (sender.kind === 'terminal' && holder.kind === 'mobile') {
    return 'Mobile holds lock. Press Ctrl+\\ Ctrl+T to force takeover.';
  }
  if (sender.kind === 'mobile' && holder.kind === 'terminal') {
    return 'Terminal holds lock. Tap "Take over" to force takeover.';
  }
  return `Lock held by another ${holder.kind} device.`;
}
```

```typescript
export interface IndexedEvent {
  uuid: string;
  offset: number;
  byteLen: number;
  duplicate: boolean;
}

export type RecordResult =
  | { kind: 'new' }
  | { kind: 'duplicate'; firstOffset: number };

export class EventIndex {
  private readonly entries: IndexedEvent[] = [];
  private readonly firstByUuid = new Map<string, number>(); // uuid → offset of first occurrence
  private dupCount = 0;

  record(uuid: string, offset: number, byteLen: number): RecordResult {
    const first = this.firstByUuid.get(uuid);
    if (first !== undefined) {
      this.entries.push({ uuid, offset, byteLen, duplicate: true });
      this.dupCount++;
      return { kind: 'duplicate', firstOffset: first };
    }
    this.firstByUuid.set(uuid, offset);
    this.entries.push({ uuid, offset, byteLen, duplicate: false });
    return { kind: 'new' };
  }

  locate(uuid: string): IndexedEvent | null {
    const first = this.firstByUuid.get(uuid);
    if (first === undefined) return null;
    return this.entries.find((e) => e.uuid === uuid && !e.duplicate) ?? null;
  }

  *iterateAfter(uuid: string): Iterable<IndexedEvent> {
    let started = false;
    for (const e of this.entries) {
      if (started) {
        if (!e.duplicate) yield e;
        continue;
      }
      if (e.uuid === uuid && !e.duplicate) started = true;
    }
  }

  get size(): number {
    return this.entries.length;
  }

  get duplicates(): number {
    return this.dupCount;
  }
}
```

```typescript
import { EventEmitter } from 'node:events';
import * as fs from 'node:fs/promises';
import * as fsSync from 'node:fs';

export interface JsonlWatcherOptions {
  path: string;
  pollMs?: number;
}

export interface JsonlEvent {
  uuid: string;
  raw: string;
  offset: number;
  byteLen: number;
}

/**
 * Tail-watches a JSONL file. Emits 'event' for each complete line.
 * Holds back any partial trailing bytes (torn write) until the line completes.
 */
export class JsonlWatcher extends EventEmitter {
  private readonly path: string;
  private readonly pollMs: number;
  private offset = 0;
  private partial = '';
  private closed = false;
  private timer: NodeJS.Timeout | null = null;

  constructor(opts: JsonlWatcherOptions) {
    super();
    this.path = opts.path;
    this.pollMs = opts.pollMs ?? 50;
  }

  async start(): Promise<void> {
    try {
      const stat = await fs.stat(this.path);
      this.offset = 0;
      // Catch up on existing content
      if (stat.size > 0) await this.poll();
    } catch {
      // File may not exist yet; that's fine — the daemon may create it later.
    }
    this.scheduleNext();
  }

  async close(): Promise<void> {
    this.closed = true;
    if (this.timer) clearTimeout(this.timer);
  }

  private scheduleNext(): void {
    if (this.closed) return;
    this.timer = setTimeout(() => {
      void this.poll().finally(() => this.scheduleNext());
    }, this.pollMs);
  }

  private async poll(): Promise<void> {
    let stat;
    try {
      stat = await fs.stat(this.path);
    } catch {
      return;
    }
    if (stat.size <= this.offset) return;

    const fd = await fs.open(this.path, 'r');
    try {
      const len = stat.size - this.offset;
      const buf = Buffer.alloc(len);
      await fd.read(buf, 0, len, this.offset);
      const text = this.partial + buf.toString('utf8');
      const absoluteOffsetOfChunkStart = this.offset - Buffer.byteLength(this.partial, 'utf8');
      let lineStartInText = 0;

      for (let i = 0; i < text.length; i++) {
        if (text.charCodeAt(i) === 0x0a /* \n */) {
          const line = text.slice(lineStartInText, i);
          const lineByteLen = Buffer.byteLength(line, 'utf8') + 1; // include \n
          const lineStartByte =
            absoluteOffsetOfChunkStart + Buffer.byteLength(text.slice(0, lineStartInText), 'utf8');
          this.consumeLine(line, lineStartByte, lineByteLen);
          lineStartInText = i + 1;
        }
      }

      this.partial = text.slice(lineStartInText);
      this.offset = stat.size;
    } finally {
      await fd.close();
    }
  }

  private consumeLine(line: string, offset: number, byteLen: number): void {
    if (line.length === 0) return;
    let parsed: { uuid?: unknown };
    try {
      parsed = JSON.parse(line) as { uuid?: unknown };
    } catch (err) {
      this.emit('parseError', { line, offset, error: err });
      return;
    }
    const uuid = typeof parsed.uuid === 'string' ? parsed.uuid : null;
    if (uuid === null) {
      this.emit('parseError', { line, offset, error: new Error('missing uuid') });
      return;
    }
    const ev: JsonlEvent = { uuid, raw: line, offset, byteLen };
    this.emit('event', ev);
  }
}
```

> **Note on the `fs` shape:** We import `node:fs/promises` for normal async ops and rely on the test's `fsSync` to simulate writer behavior. The watcher itself uses only async `fs.stat` / `fs.open`.

```typescript
import { EventEmitter } from 'node:events';
import { JsonlWatcher, type JsonlEvent } from './jsonlWatcher.js';
import { EventIndex, type IndexedEvent } from './eventIndex.js';

export interface ReplayEngineOptions {
  path: string;
  pollMs?: number;
}

export class ReplayEngine extends EventEmitter {
  private readonly watcher: JsonlWatcher;
  private readonly index = new EventIndex();

  constructor(opts: ReplayEngineOptions) {
    super();
    this.watcher = new JsonlWatcher({ path: opts.path, pollMs: opts.pollMs });
    this.watcher.on('event', (e: JsonlEvent) => this.onLine(e));
    this.watcher.on('parseError', (err) => this.emit('parseError', err));
  }

  async start(): Promise<void> {
    await this.watcher.start();
  }

  async close(): Promise<void> {
    await this.watcher.close();
  }

  locate(uuid: string): IndexedEvent | null {
    return this.index.locate(uuid);
  }

  *replayAfter(uuid: string): Iterable<IndexedEvent> {
    if (this.index.locate(uuid) === null) {
      throw new Error('cursor-invalidated');
    }
    yield* this.index.iterateAfter(uuid);
  }

  private onLine(e: JsonlEvent): void {
    const r = this.index.record(e.uuid, e.offset, e.byteLen);
    if (r.kind === 'duplicate') {
      this.emit('duplicate', { uuid: e.uuid, firstOffset: r.firstOffset, secondOffset: e.offset });
      return;
    }
    this.emit('broadcast', { uuid: e.uuid, raw: e.raw });
  }
}
```

- **Per-family rotation serialization (spec §7 critical implementation requirement):** concurrent refresh calls on the same family cannot mint divergent successors — implementation uses per-family async-mutex OR storage-layer atomic compare-and-swap. Verified by stress test: spawn N concurrent `auth-token-refresh` calls with the same `R0` token; assert exactly one new successor minted, others either return the same successor (idempotent) or trigger divergence revocation
- **Server-clock authoritative grace window (spec §7):** grace expiry uses daemon clock only; client `timestamp` field does NOT extend grace. Verified by unit test passing client timestamps in the future / past — server still revokes after `graceExpiry = serverNow + 30s`
- **Stateful access-token revocation check (spec §7):** every authenticated envelope's access-token hash is checked against the family's revocation state on each request — once a family is revoked, in-flight access tokens fail at the next envelope (not at 1h expiry). Verified by integration test: issue access token → revoke family out-of-band → next envelope rejected with `auth-error`. Increments `tether_access_token_revoked_rejections_total`

**Key risks:**
- Idempotency under concurrent grace-window calls — use per-family async-mutex
- Clock skew handling: ±5min documented; warning log if skew > 60s
